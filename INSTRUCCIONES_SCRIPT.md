# 🎮 INSTRUCCIONES PARA USAR EL SCRIPT

## 📁 ARCHIVO: `death_ball_working_script.lua`

### 🚀 PASOS PARA USAR:

1. **Abre el archivo**: `death_ball_working_script.lua`
2. **Selecciona TODO** el contenido (Ctrl+A)  
3. **Copia** el código (Ctrl+C)
4. **Abre Velocity** (o tu executor)
5. **Pega** el código en Velocity (Ctrl+V)
6. **Entra a Death Ball** en Roblox
7. **Presiona Execute** en Velocity
8. **¡LISTO!** - Debe aparecer la GUI

### ✅ QUÉ DEBES VER:
- 🔔 **Notificación**: "✅ Script cargado!"
- 🖥️ **GUI roja** en la esquina izquierda
- 📝 **En consola**: "Death Ball Ultra iniciado correctamente!"

### 🎮 CONTROLES:
- **F** = Parry Manual
- **Q** = Toggle Auto Parry  
- **E** = Toggle Follow Ball
- **R** = Toggle AI Movement
- **G** = Cambiar velocidad (16/50)
- **RightShift** = Mostrar/Ocultar GUI

### 🏐 FUNCIONES EN LA GUI:
- **Auto Parry** - Hace parry automáticamente
- **Follow Ball** - Sigue la pelota
- **AI Movement** - Movimiento inteligente
- **Auto Skill** - Habilidades automáticas  
- **Infinity Dash** - Velocidad infinita
- **Orbit Ball** - Orbitar la pelota
- **Manual Parry** - Botón para parry manual

### 🚨 SI NO FUNCIONA:
1. Verifica que Velocity esté "Injected"
2. Asegúrate de estar EN Death Ball
3. Prueba cerrar y reabrir todo
4. Dime exactamente qué error aparece

¡El script está completo y listo para usar!
